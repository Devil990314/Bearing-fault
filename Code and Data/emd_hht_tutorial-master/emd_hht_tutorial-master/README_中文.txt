# 利用经验模式分解进行时频分析

本教程介绍两个概念：
1) 利用瞬时频率估计进行高分辨率时频分析
估计
2) 将复杂信号分离为较简单的振荡成分
成分。

结合这些步骤，我们就能以比大多数标准方法更高的分辨率计算时频图。
时频图。

实地考察工具箱

要运行本教程，您需要运行 matlab R2018a 或更高版本的
信号处理工具箱。

在旧版本中不可用。
