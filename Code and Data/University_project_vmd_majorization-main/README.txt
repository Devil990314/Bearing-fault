使用vmd算法对含有噪声的图像信号进行分解，去除掉噪声信号，将剩余信号合成，得到去噪声图像。分别使用alo、ao、ga、gwo、mpa、spo、woa算法对vmd算法中的参数进行优化，实现快速、准确的完成图像信号的分解。

ALO（Ant Lion Optimizer）：A Ant Lion Optimizer（蚁狮优化算法）是一种基于自然界蚁狮捕食过程的优化算法。该算法使用蚁狮的捕食行为来调整解空间中的候选解，以寻找最佳解。ALO的思想是模拟蚁狮的捕食行为，其中蚂蚁沿着坑道移动，而蚁狮隐藏在坑底等待捕食。

AO（Astronomical Optimization）：Astronomical Optimization（天文学优化算法）是一种基于天文学原理的优化算法。它使用了一些天文学中的概念，如引力、质心和星系的模拟来实现全局搜索和局部搜索，以找到最优解。

GA（Genetic Algorithm）：遗传算法是一种基于生物遗传学原理的优化算法。它使用种群中的个体之间的交叉和变异操作来搜索最优解。GA通过迭代进化过程中的选择、交叉和变异来搜索最优解，通常用于解决复杂的优化问题。

GWO (Grey Wolf Optimizer)：灰狼优化算法是受灰狼行为启发的一种优化算法，通过模拟灰狼群体的协作行为来优化问题。它具有简单、易于实现以及高效的特点。

MPA (Moth-flame Optimization Algorithm)：MPA算法：蛾火优化算法是通过模拟蛾群中蛾火的行为来进行优化的算法。蛾火会受到光源的吸引，算法通过模拟蛾火的飞行和跳跃行为来搜索最优解。

SPO (Shuffled Frog Leaping Algorithm)：SPO算法：洗牌蛙跳算法是一种基于蛙群行为的优化算法。它通过模拟蛙群中蛙的跳跃行为来进行搜索，其中蛙的位置会通过洗牌操作进行更新。

WOA (Whale Optimization Algorithm)：WOA算法：鲸鱼优化算法是一种受鲸鱼觅食行为启发的优化算法。鲸鱼会通过模拟尾巴的摆动来搜索食物，算法通过模拟鲸鱼的行为来进行优化。
